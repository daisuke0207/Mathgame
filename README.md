# MathGame  

<br>

# 概要
足し算をして繋げていく数字パズルゲーム。  
指定されている数字とイコールになるように数字を繫げてパズルを消していき、タイムを伸ばします。画面の上部にある線に約5秒間パズルが触れていたらゲーム終了です。  
<br>

# 制作背景
開発の練習として作成しました。ツムツム風のパズルゲームにした理由は、ネットにツムツム風のゲームの情報があり、勉強がしやすいと考えたからです。
